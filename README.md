# Weather Lookup

A small page that shows the latest temperature and wind for a handful of popular cities.

## How to use it
- The page is on Gitpages, i provided a link separately.
- Type one of these city names: Nairobi, London, New York, Tokyo, Paris, or Sydney.
- Click **Search** (or press Enter) to see the current conditions.
- If something goes wrong, a clear message will tell you what to try next.
